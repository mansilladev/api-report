MyAppBuilder.com announces release of it's File Sharing App and Service. MyAppBuilder.com is a pioneer in creating mobile apps,. MyAppBuilder is a cloud service that provides cross platform app publishing including iOS, Android and HTML5.

File Sharing App/Service allows MyAppBuilder clients to create their own File Sharing App. MyAppBuilder has extensive API to access MyAppBuilder Platform as well as specific calls to the File Sharing API’s. While the platform is impressive, the best part is MyAppBuilder release full functioning File Sharing Apps. File Sharing App is turn key and can be instantly deployed. Roger Lichfield CEO of MyAppBuilder states “I am excited to see all the varieties and uses for this application, I feel software is about freedom, we release App as Open Source”.

File Sharing App includes several features right out of the box including: iOS App, Android App, URL Shortener, Share Files Feature, File Revision Control, Archive Downloads, Browser Based, Intuitive File System separated into Files and Folders. MyAppBuilder offers the Apps on Android App Store and Apple App Store. Source Code can be downloaded from MyAppBuilder Site.
