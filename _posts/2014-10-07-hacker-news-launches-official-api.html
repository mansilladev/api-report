The popular tech news aggregation site Hacker News just launched an API, and acknowledging the appetite of its very tech savvy audience, has also ensured that the API is real-time. The YCombintaor backed Hacker News has partnered with real-time framework Firebase to launch an API layer that developer can use to consume, and engage with the Hacker News community in real-time.

In the announcement for the new API, HN acknowledges that they launched the API in response to there being "a lot of apps and projects out there that rely on scraping the site to access the data inside it”, so HN decide it was "best to release a proper API and give everyone time to convert their code before we launch any new HTML”.

HN chose to partner with Firebase to give "developers near real-time access to our data”, but also at the same time provide them with "well designed, easy to use SDKs for iOS, Android and the Web, in addition to a REST API”. Serving both the needs of HN in managing their own UI evolution, but also serving the needs of their existing community.

You can access the real-time HN API at https://hacker-news.firebaseio.com, and direct questions and feedback to api@ycombinator.com.
