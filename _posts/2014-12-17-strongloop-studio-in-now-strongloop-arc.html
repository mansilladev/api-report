The industry's first graphical UI for managing each step of the API development lifecycle using Node.js In case you missed our announcement yesterday, StrongLoop Studio has been renamed to StrongLoop Arc and besides the API composition features you are already familiar with, we've added tools for building, profiling and monitoring Node apps.

To learn about the name change and the new features we've added, check out StrongLoop CTO Al Tsang's blog and in-depth video demo.

How do I upgrade from Studio to Arc?

For those of you currently running previous releases of Studio Beta, there's two things to be aware of:

  • Install Arc by simply updating via 'npm -g install strongloop'
  • Now, instead of running 'strong-studio' you run 'slc-arc'
