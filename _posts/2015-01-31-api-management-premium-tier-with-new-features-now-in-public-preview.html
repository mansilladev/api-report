The new Premium tier of API Management simplifies massive scale (each unit includes 1 billion API calls per month) and high availability. Geo-distribution provides a turnkey way for you to host an API Management instance in multiple Azure datacenters with automatically synchronized configuration around the world. Added VPN connectivity means you can connect the API Management proxy to your Azure Virtual Network (including ExpressRoute) to help provide security and performance when tackling hybrid scenarios.
