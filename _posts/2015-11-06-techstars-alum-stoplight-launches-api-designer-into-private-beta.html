StopLight, an early leader in the nascent API Management market, has released API Designer into private beta. The company’s powerful editor provides the fastest and most frictionless way for teams to design, manage and collaborate on APIs in real-time - enhancing productivity. Developers can visit https://stoplight.io today for early access to the API Designer.

APIs are critical to any business operating in today’s digital environment. Every mobile app, IoT device, and web service, is made possible by API technology. StopLight’s solution aims to eliminate wasted developer hours and bugs, improve team communication, streamline workflows, and improve the overall quality of the modern API. Net results include better products and more satisfied teams.

According to Keith Casey, co-author of “A Practical Approach to API Design”, “Building a house without a plan is foolish but we've done the same with APIs for years. If we put just a little thought into design, we can create APIs that work well now and into the future.”

API Designer’s core features include:

  • Collaborative discovery & documentation
  • Import / export to existing API specs like Swagger
  • Validation
  • Mocking and transformations
  • A variety of simple visual interfaces
  • Prism API Proxy integration

“Developers are one of a company's most expensive resources. With API Designer’s focus on debugging, documentation, and collaboration, we are helping development teams do more in less time, improve the quality of their APIs, and shift focus back to innovation,” said Marc MacLeod, Founder and CEO of StopLight.

Prism, a lightweight dependency-free proxy, sits behind API Designer and enables features like API mocking, validation, and transformation. Prism makes it possible for teams to truly work independently but collaboratively, vastly increasing developer productivity and efficiency.    

Developers can visit https://stoplight.io for early access to the API Designer free private beta.

About
StopLight, headquartered in Austin, TX, is an API workflow management and collaboration tool. The company was founded in 2015 and is a graduate of Techstars. More info available at Stoplight.io or contact press(at)stoplight(dot)io.
