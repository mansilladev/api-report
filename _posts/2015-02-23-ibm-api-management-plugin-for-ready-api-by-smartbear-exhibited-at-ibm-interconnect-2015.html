SmartBear Software, the choice of more than two million software professionals for building and delivering the world’s best applications, is exhibiting at IBM InterConnect, Monday, February 23 – Thursday, February 26, 2015 in Las Vegas, including its IBM API Management plugin for Ready! API that allows users to import APIs directly from the IBM API Management developer portal as well as publish an API to the IBM platform. Ready! API is the first fully integrated, extensible and affordable platform to help development, testing and operations teams build reliable, scalable and secure APIs.

“Whether your API is available to external development teams or internal product teams, developers need a robust API readiness plan for this business critical asset, just as you would prepare a product readiness plan for a standard application”

and AlertSite UXM™

“Whether your API is available to external development teams or internal product teams, developers need a robust API readiness plan for this business critical asset, just as you would prepare a product readiness plan for a standard application,” said Lorinda Brandon, Director, API Partner Development at SmartBear. “IBM and SmartBear created an integration that brings the two technologies together, allowing teams to focus on API quality in addition to API operations. The plugin provides functionality for both the API consumer and API provider, regardless of whether you are using SOAP or REST as your protocol.”

SmartBear is also demonstrating Collaborator, the only tool available that allows peer review of source code, user stories, test plans and more with support for .docx, .xlsx and .pdf files. Collaborator is tightly integrated into IBM’s Rational Team Concert (RTC). The direct integration allows users to quickly create reviews based on work item state changes. Collaborator also supports Rational ClearCase and Synergy. For more information, visit:http://blog.smartbear.com/code-review/enhanced-collaboration-integration-with-ibms-rational-team-concert/. On Twitter, follow @CollaboratorSB.

AlertSite UXM by SmartBear helps application development and operations teams to deliver application performance and availability for Web, API and mobile customer experiences. AlertSite UXM is one platform to proactively monitor APIs, mobile, Web and SaaS applications from the end user’s perspective. API functional test scripts can be imported directly from SoapUI NG Pro to monitor APIs in production from 80+ global locations. For a free trial of AlertSite UXM, visit: http://alertsite.com/free-trial/synthetic-and-real-monitoring.html. On Twitter, follow @AlertSiteUXM.
