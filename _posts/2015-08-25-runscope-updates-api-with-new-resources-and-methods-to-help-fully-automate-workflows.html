Today, Runscope has announced the official release of their newly improved and updated Runscope API. This update allows customers to fully integrate Runscope into their workflow, and manage tests, environments and schedules programmatically. All of the test building and management functionality delivered in the Runscope dashboard UI can now be fully managed via the API.

Learn more in Runscope's announcement blog post. Or visit the API explorer to traverse resources and try live API calls.
