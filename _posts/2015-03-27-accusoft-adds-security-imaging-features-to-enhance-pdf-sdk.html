Accusoft today announces the addition of enhanced security features, Page Rotation, and User Space Resolution to PDFXpress v7, a comprehensive PDF library used for building feature-rich applications.

Already known for providing users more control over PDF documents, the new features strengthen that reputation by boosting security and offering better image manipulation.

Expanded Security Features
Expansion of support for security features beyond basic password protection broadens the versatility of the product to those that work with secure documents. Included within this PDF SDK are customizable security settings, permissions and extensive encryption methods.

Customizable Security Settings - Developers can read, access, edit and remove the security settings of a secure document including Permissions, Encryption Method, User and Owner Passwords. They also have the ability to add security settings to a non-secure document.

Permissions - Thorough, defined permissions allow for document modification, text annotation or modification, and copying and/or extracting text and graphics for individual users or in support of users with disabilities or other purposes. It also allows for document assembly, faithful digital and low-level printing, and filling in interactive form fields.

Encryption Methods

  • 40-bit RC4, Acrobat 3 and later 
  • 128-bit RC4, Acrobat 5 and later 
  • 128-bit RC4, Acrobat 6 and later 
  • 128-bit AES, Acrobat 7 and later 
  • 256-bit AES, Acrobat 9 and later

Page Rotation Capabilities
For further flexibility, the ability to rotate a page has been added. Users now have the ability to rotate a page in a document, adding further flexibility and control.

Rendering PDFs Beyond the 72dpi Default
PDFXpress now provides an API that allows developers the ability to retrieve the resolution of the page, this is important for pages that don’t use the 72 dpi default. This empowers the app to correctly display the image and for image editing features like cropping and zooming to interact appropriately with the rendered image.

“PDFXpress provides the tools the development community needs to create high-functioning, competitive PDF applications,” said Accusoft President Jack Berlin.

Accusoft plans to continue the enhancement of the PDF SDK to provide developers with superior PDF technology and continue to drive it to the forefront of the market.
