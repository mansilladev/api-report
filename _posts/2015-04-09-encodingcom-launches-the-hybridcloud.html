Encoding.com has introduced the Encoding.com HYBRIDCLOUD, a cloud orchestrated, process anywhere solution that empowers broadcasters, original content programmers and video platform providers to better manage the scale and complexities of the fast-changing digital video ecosystem. 

Combining the best benefits afforded by cloud, software-as-a-service and on-premise video infrastructure, video publishers can cost effectively burst from on-premise to cloud platforms to meet real-time volume demand while minimizing large file transport and maintaining security requirements.  Customers integrate the Encoding.com API and gain access to an ever-increasing feature set deployed in both on-premise and public cloud architecture. 

"The ability to process massive files locally and burst to global compute resources when necessary, options for CAPEX and OPEX cost models, cloud orchestration for media processing in multiple data centers, and the ability to quickly support new workflow requirements make the Encoding.com HYBRIDCLOUD the ideal solution for the next-generation digital video supply chain," said Gregg Heil, CEO at Encoding.com.

The Encoding.com HYBRIDCLOUD's job orchestration layer resides in the cloud enabling it to intelligently manage all available compute resources and software in customer private clouds in any of Encoding.com's 11 MPAA certified public cloud processing centers.

Key highlights:

  • Scale- Burst instantly from on-premise to cloud to meet real-time demand.
  • Speed and Security- Process large files within customer's network reducing transport. High-value content stays behind the customer's firewall.
  • Managed- 30+ transcoding engines maintained by Encoding.com including licenses, updates, patches, and hotfixes without workflow disruption.
  • Flexibility- Integrate Encoding.com's powerful API with existing media management systems, automating end-to-end workflows.
  • Cost- Fixed CAPEX annual licensing and variable OPEX models to maximize cost efficiency.
