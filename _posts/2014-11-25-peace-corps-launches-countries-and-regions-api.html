The Peace Corps has recently launched the Countries & Regions API providing read-only access and currently supports JSON data format. Developers, designers, engineers, and partners are all encouraged to leverage the Peace Corps Countries & Regions API to build innovative tools and/or creatively share this data.

The Peace Corps Countries dataset includes all countries Peace Corps Volunteers have served in and are currently serving in. The Peace Corps Regions dataset includes all the Peace Corps regions. Individual users (per IP) are limited to 15 requests per second. Developers, designers, engineers, and partners are all encouraged to leverage the Peace Corps Countries & Regions API to build innovative tools and/or creatively share data.

The Peace Corps Countries & Regions API can be access via the organizations Github repository at https://github.com/PeaceCorps/Countries-Regions-REST-API.
