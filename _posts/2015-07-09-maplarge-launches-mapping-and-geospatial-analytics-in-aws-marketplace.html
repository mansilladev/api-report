Today, MapLarge announces the launch of the MapLarge Mapping Engine on Amazon Web Services (AWS) via a listing in AWS Marketplace. The listing in AWS Marketplace provides quick and easy access to the MapLarge Mapping Engine for customers around the globe.

MapLarge Mapping Engine provides a unified platform for organizations to perform collaborative, geospatial analysis, visualization and publishing. MapLarge Engine users can access a self-service geospatial business intelligence application to perform data discovery and reporting workflows with their data. Enterprise deployments can integrate the MapLarge Mapping Engine into their existing workflows using an extensive set of APIs.

AWS Marketplace allows customers to browse, purchase and download software via 1-Click® deployment from a wide selection of products offered on an hourly, monthly, or annual basis. Customers can visit the MapLarge product page in AWS Marketplace to purchase and deploy an annual license on AWS.

“AWS Marketplace provides an enabling channel for innovative organizations to quickly evaluate, test, and deploy the MapLarge Mapping Engine on AWS,” said Lynwood Bishop, President and CEO of MapLarge.

“We’re pleased that MapLarge Mapping Engine is now available in AWS Marketplace, helping to lower the barriers for organizations to rapidly adopt this geospatial technology,” said Barry Russell, Head of Global Business Development for AWS Marketplace, Amazon Web Services, Inc.

About MapLarge: MapLarge provides a fully customizable, elastic, cloud based platform for geospatial analytics and visualization. MapLarge is an industry leader in mapping large quantities of data to large audiences with cutting edge data processing technology that scales in the cloud. User-driven, real-time, interactive visual exploration of Big Data is available through any web browser. All of the MapLarge apps use open standards for communication and are drop in compatible with ESRI, Google, LeafletJS, OpenLayers and OpenStreetMap. Location based transit insights are available with the MapLarge Drive Time application and can be bundled with detailed demographics for nationwide marketing and site location analytics.

For more information please visit our website http://MapLarge.com.
