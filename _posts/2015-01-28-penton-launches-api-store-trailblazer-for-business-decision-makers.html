Penton, North America's largest privately held professional information services company, today introduced a new API store, continuing its focus on user workflow tools.

"This is an exciting time to be in the data and information industry. We are aggressively building on our growth in data and digital with new API product introductions," said Richard Stanton, Chief Data Officer, Penton. "We successfully launched eight data products and see our API marketplace as a natural progression to our offerings. This strategy allows us to create fast, reliable APIs to build a new revenue channel and spur innovation. As trailblazers in the information services space, one of our biggest opportunities is to deliver content and data to our audiences anytime, anywhere through APIs."  

Price Digests API for Trucking and Automotive
The Price Digests API for the trucking and automotive industries (http://api.penton.com/pricedigests) is the first tool to be released through Penton's API platform. It offers vehicle identification number (VIN) driven valuation and specification data for thousands of consumer equipment models. Targeted to insurance, revenue, transportation and driver services departments as well as finance organizations, this enhanced offering has been created based on 100+ years of expertise delivering accurate, comprehensive information for the automotive marketplace.

The Price Digests API for trucking and automotive offers access to original and used valuations along with basic specifications for thousands of consumer equipment models including, commercial trucks, automobiles, powersport, trailers and recreational vehicles. It also offers information for return models using either VIN or make and model and adjusts for age and usage to ensure accurate valuations. 

Key features of the Price Digests API for the trucking and automotive industries:

  • Achieve real-time data exchange between platform and customer applications
  • Direct access to production database; no file importing or updating required.
  • Easy integration into current applications.
  • Easy Change Management. Only one place in code when adding new data element.
  • Easily adaptable attributes such as values, specs or some combination.  Solution is targeted to the customer and that is easily managed.
  • Standard protocol (SOAP/XML) that can be integrated into any platform or environment.
  • Enhanced visibility and reporting into data consumption.

The Price Digests API for the trucking and automotive industries is valuable to the insurance market for underwriting, coverage and claims; to transportation and driver services departments for VIN intelligence as well as property and sales tax; and finance organizations rely on the data for vehicle worth for financing and trade-ins.
