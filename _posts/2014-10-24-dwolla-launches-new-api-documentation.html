A few months ago we discovered TripIt’s awesome documentation framework, Slate, and fell in love with its style, ease of use, and transparency.

Since then, we’ve been rewriting our developer documentation, and now we’re excited to launch the new Dwolla API docs, powered by Slate.

The endpoint documentation has been removed from our Developer’s Portal’s sidebar, and any existing links now redirect to the new docs.

Today we're excited to launch the new Dwolla API docs powered by Slate—a dynamic and transparent documentation framework for developers.

Head on over and take a look through our libraries and sample code, and feel free to leave us some feedback!
