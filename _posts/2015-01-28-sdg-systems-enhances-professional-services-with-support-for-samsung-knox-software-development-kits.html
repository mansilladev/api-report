SDG Systems, a leader in system software and operating system support for Android on enterprise platforms, recently expanded its enterprise software development services with support for Samsung KNOX standard and customization SDKs. The SDKs provide application programming interfaces (APIs) to developers. The KNOX standard APIs provide the basic functionality required for enterprise applications, including improved device control, security and management. The customization APIs provide features to create a powerful user experience and to control device configuration. These APIs include the ability to deploy applications in “sealed” or kiosk mode.

The KNOX programming services may be combined with SDG’s other software solutions, like the Enterprise Platform Extensions for Barcode Scanning (EPX-B). EPX-B provides a modular barcode scanning solution for Android that can utilize different types of barcode scanners (camera, USB, Bluetooth) with a single API. The EPX-B software is available today in the Google Play store and is highlighted in Samsung’s SEAP solution showcase.

“SDG Systems is pleased to be part of the Samsung SEAP and KNOX partner programs, offering our EPX software and KNOX development services,” said Todd Blumer, President of SDG Systems. “Samsung is a leader in enterprise functionality for Android, so it is a complementary extension to the work that we have been doing for the past few years. Being a customization partner, SDG can assist in the deployment of Samsung-based solutions, including acquiring the necessary commercial licensing.”

To demonstrate the EPX-B software, and Samsung KNOX customization showing a sealed mode application, the company has posted a YouTube video. To learn more about Samsung KNOX, please visit SDG Systems' blog orSamsung's KNOX website. Please contact SDG Systems for a full demonstration or for additional information. Other services offered by SDG Systems include Android porting, systems integration, device driver implementation, and application development.
