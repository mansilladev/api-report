We have released our API for the NIH 3D Print Exchange, thanks to help from Gray Brooks and the 18F team. General information can be found at http://3dprint.nih.gov/developer, and code and documentation can be found on the NIAID GitHub: http://niaid.github.io/3dpx_api/. READ API available now, WRITE API is in development. 

The NIH 3D Print Exchange is an open, comprehensive, and interactive website for searching, browsing, downloading, and sharing biomedical 3D print files, modeling tutorials, and educational material. We are looking forward to seeing how our API will be used to bring added value to our database through integration with other sites.

We hope to see this listed on api.data.gov soon. Any and all feedback is welcome.
