Wurlie is the most popular URL shortening script available today, offering a quick and easy solution to entrepreneurs and businesses to create their own URL shortening service. Initially released in 2010 by MFScripts, Wurlie has quickly grown into the feature rich PHP website script it is today, with over 800 webmasters using it to power their websites. A few of the notable features available for the URL shortening script include; a fully responsive design, advertisement integration, an extensive admin area, multiple URLs and more.

This month Wurlie has officially released version 3.4 of its URL shortening script. The latest version delivers additional functionality as well as hugely boosting it's revenue earning capabilities. The new features are as follows:

  • Added paid accounts option. Offer account upgrades via PayPal which hide ads. 
  • Full support for case sensitive short urls. 
  • Bulk delete users/urls in admin area. 
  • Added thumbnail of original site on stats pages. 
  • Added tools section to list Bookmarklet, API & HTML Convert. 
  • Added a contact us form, with optional captcha. 
  • Upgraded reCaptcha to reCaptcha2. 
  • Optional private Url statistics via the account settings.

Wurlie CEO Adam Wilson said, "We've been overwhelmed by the feedback from our customers regarding Wurlie short URL script. With the feedback we also received a lot of ideas on how to improve the script and so are pleased to release version 3.4."
