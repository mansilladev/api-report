CloudRail has launched CloudRail for Cloud Storage Services. By simply integrating the CloudRail adaptive API, software developers can seamlessly connect their application with other services faster and easier than ever before. This one-to-many approach works because the CloudRail interface can download special API definitions to automatically adapt to other systems. This dramatically speeds up the development process and allows developers to focus on building stunning software instead of wasting time for integrations.

“Convenience and fast development. The best facility for integrating cloud services through a single API with entire security” stated John Haden of TrooMobile, a leading web and mobile app development company that was part of CloudRail¹s early adopter program.

Core features and features of CloudRail include:

  • Easy integrations - connect applications to more services, thus increasing customer reach
  • Faster and cheaper - a single API for connecting to all cloud storage providers
  • Automatic API updates - the CloudRail interface updates with providers change API. Developer maintenance cost go to zero
  • No middleware - conversion happens in the client, so the data never passes through CloudRail. It is a peer-to-peer connection with the provider.

The product is available immediately at http://www.cloudrail.com. Developers can get started for free and grow into the Premium and Enterprise offerings with scale.
