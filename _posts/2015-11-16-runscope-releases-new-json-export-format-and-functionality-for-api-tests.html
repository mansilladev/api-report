Runscope has released the new Runscope Export Format, a JSON representation for API tests, along with import and export functionality that allows users to export entire suites of API tests in one click. These new features are instrumental for streamlining API test management and integrating Runscope API testing and monitoring into existing software development and deployment practices.

Learn more in Runscope's announcement blog post.
