Kevin Dunglas, a core developer of Symfony, the popular full-stack PHP application framework, has released a new API-first framework dubbed API Platform. The concept of taking an API-first approach to development isn't new. Kin Lane wrote a blog post about this the very topic back in 2014. However, there hasn't been a lot of community projects/frameworks that directly facilitate an API-first approach. Out of the box, API Platform supports hypermedia APIs, JSON-LD and Hydra. Dunglas built and bundled several useful tools, including docs generation, Schema.org support, entity auto-generation, and more.

Neil Mansilla from Runscope gave API Platform a try, and wrote up a summary in a blog post.
